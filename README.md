## installation
- Fisrt clone repo:
```
$ git clone https://github.com/sina-yeganeh/kid-keeper.git
```

- Then run `installer.sh`:
```
$ ./installer
```

- And after run `main.py` with this command:
```
$ python3 main.py
```
